  • input type="text" 文本框，用于文本的输入
  • input type="password" 密码输入框，用于密码的输入，与文本框的区别是，输入的内容不能被用户看到
  • input type="radio" 单选框，通过name控制类别，name想通的radio只能被选择一个
  • input type="checkbox" 复选框，用过name控制类别，但是可以选择多个
  • input type="button" 按钮
  • input type="submit" 提交按钮，可以将表单数据提交至网站后台


css选择器共分为四类

元素选择器，直接用表现指定元素

类选择器，在style代码前加一个.,在标签中通过class进行匹配

id选择器，在style代码名前加一个#,在标签中通过id进行匹配建议所有代码都使用class选择器，id选择器只做了解即可

通配符选择器，通配符选择器可以找到html元素中的所有元素,style代码前用*
